# Trove

A free, standards-based digital credential wallet with an AI advisor that turns
verified credentials into career and education guidance.

## Stack
Next.js (App Router, TypeScript) · Tailwind CSS · Supabase · Vitest

## Development
```bash
npm install
npm run dev     # http://localhost:3000
npm test        # run the test suite
```

See `docs/superpowers/` for the design doc and implementation plan.
